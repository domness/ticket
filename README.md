Ticket
======

Ticket is a simple Rails application that allows to you set up your own simple Kanban board anywhere such as Heroku.

Dependencies:

- Ruby v2.0.0
- Rails v4.0
- Postgres
- PhantomJS

### Features

- Multiple columns: Backlog, Current, Complete
- T-shirt size items
- Assign items to users within the system
